# BoardView

Виджет для просмотра платы.

## Пример

1. Установите необходимые зависимости:

   ```bash
   python -m pip install --upgrade pip
   python -m pip install -r requirements.txt
   ```

2. Запустите пример:

   ```bash
   python examples/example_1.py
   ```

3. Запустите тесты:

   ```bash
   python -m unittest discover -v
   ```
